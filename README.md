## Features

The Telegram adapter allows for reading and writing of messages from the [Telegram](https://telegram.org/) messaging platform using [`python-telegram-bot`](https://python-telegram-bot.readthedocs.io/).

[More information is available in our wiki](https://github.com/csp-community/csp-adapter-telegram/wiki)

## Chat Framework

[`csp-bot`](https://github.com/Point72/csp-bot) is a framework for writing cross-platform, command oriented chat bots.

## Installation

Install with `pip`:

```bash
pip install csp csp-adapter-telegram
```

Install with `conda`:

```bash
conda install csp csp-adapter-telegram -c conda-forge
```
